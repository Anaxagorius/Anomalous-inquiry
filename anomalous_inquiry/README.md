
# Anomalous Inquiry

Neutral, documentary-style research platform built with Rust + Axum.

## Features
- Static articles (Markdown)
- Reflections journal (admin)
- Moderated anonymous comments
- Suggestions inbox
- CE1–CE5 archive & timeline
- PDF export
- RSS feed
- Citation formatting (APA / Chicago)
- No frontend JavaScript

## Run
cargo run
View at http://localhost:8080
